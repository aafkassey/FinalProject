Pictionary Generator

A simple web application to store, create and reuse Pictionary templates to play the game with your friends, family and coworkers!

How to Play Pictionary

The main gist of the game is explained at this site.

To summarize, it is a simple game where the audience has to guess what the "Picturist" is trying to communicate through their drawing!

How can this site help?

This site can store game templates. The templates contain:

  • Game Title - Which is the name for the game.
  • Game Theme - The theme for the template. (Example: Christmas, 4th of July)
  • Game Rules - Rules the participant and audience will follow to play the game.
  • Game Topics - Topics are what the picturist is going to draw and is what the audience has to guess.
  • Game Hint - Hint is provided to help the audience guess what the topic is!

Create a Game

Follow below instructions:

  1. Click on Create Game on the header
  2. Fill in the information:
    1. Enter the title of the Game
    2. Enter the theme of the Game
    3. Select multiple rules from list
    4. Click on Add Topic to add a row and enter Topic and Hint
  3. Click Submit when done!

List of Games

Select a game from a list of games

Click on Game List on the header.

On this page, you will see a list of games based on when they were originally created.

Each game has an ID associated to it, that is how the system retrieves the information from the back-end database. The pages only shows the game ID, Title and Theme. For more details, click on Details button.

To delete a game, click on Delete button.

Game Details

Details for a specific Game

This page can only be reach by clicking on Details button on Game List Page.

On this page, you will see full information regarding a specific game.

Each game has a Title and Theme. The list of the rules associated with that game. Finally, a list of topics and their corresponding hints are listed out in a table view.
